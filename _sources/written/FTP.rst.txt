FTP-сервер устройства
==============================

Доступ к FTP-серверу
-------------------------

При подключении устройства к компьютеру с помощью USB кабеля в системе должно появиться 
сетевое устройство. Драйвер в операционных системах *Windows* и *Linux* должен устанавливаться автоматически.
Доступ к FTP-серверу осуществляется по адресу `ftp://169.254.241.1 <ftp://169.254.241.1>`_. 
Имя пользователя, которому разрешена запись - **admin**, пароль по умолчанию - **0000**.

Содержимое файловой системы устройства
---------------------------------------

::

   ├── download - папка для выгрузки настроек с устройства
   ├── upload - папка для загрузки настроек в устройство
   ├── modbus_mappings.cfg - файл настройки отображения внутреннего словаря адресное пространство протокола Modbus
   ├── settings1.bin - основной файл настроек
   ├── settings2.bin - резервный файл настроек
   └── www - папка, содержащая WEB-интерфейс
