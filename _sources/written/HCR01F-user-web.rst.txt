WEB-Интерфейс настройки устройства
======================================

Страница быстрой настройки
----------------------------------

Страница быстрой настройки предназначена для ускорения ввода устройства в эксплуатацию,
а так же для упрощения работы по настройке устройства в процессе работы.


Общий вид страницы настройки
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Страница настройки состоит из заголовка, блока общих параметров, блока настроек интерфейсов и протоколов, 
а так же блока управления индивидуальными параметрами устройства. 

.. figure:: /img/web/general-1.png
       :align: center
       :alt: Общий вид страницы быстрой настройки

       Общий вид страницы быстрой настройки

Раздел управления автообновлением
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Предназначена для включения/отключения автообновления с помощью соответствующего переключателя,
так же содержит информацию о текущем уровне доступа:

* *Off* - контроль доступа отключен, полный доступ
* *User* - контроль доступа включен, вход не выполнен, доступ ограничен
* *Admin* - контроль доступа включен, вход выполнен, полный доступ

.. figure:: /img/web/update.png
       :align: center
       :alt: Управление обновлением и состояние системы контроля доступа

       Раздел автообновления и состояния контроля доступа

.. attention::
   Автообновление может отключиться при потере связи с устройством. Если это произошло, необходимо вручную включить автообновление. 
   В редких случаях при ошибках связи с устройством может потребоваться перезагрузка страницы(*Ctrl-F5*).


Раздел общего описания и команд
~~~~~~~~~~~~~~~~~~~~~~~~~~

Данный раздел делится на область информации и область команд. 
Раздел информации содержит:

* Код продукта(см. РЭ)
* Уникальный номер устройства(UID)
* Версия ПО
* Версия прикладного приложения(при наличии)

Раздел команд содержит:

* поле выбора и исполнения команды
* поле ввода пароля
* поле сброса к настройкам по умолчанию

.. figure:: /img/web/commands-1.png
       :align: center
       :alt: Команды управления устройством и управление доступом

       Команды и управление доступом

Поддерживаются следующие команды: 

* Включение контроля доступа
* Отключение контроля доступа
* Сброс текущего уровня доступа
* Перезагрузка устройства

.. attention:: 
   При включенном контроле доступа исполнение команд возможно только в режиме полного доступа после ввода пароля

.. note:: 
   Для сброса к настройкам по умолчанию введите слово *load* в поле ввода и нажмите кнопку *apply*

Раздел настройки сетевых интерфейсов и протоколов
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. attention::
   Содержание области настройки интерфейсов может меняться в зависимости от варианта исполнения устройства

Настройка интерфейса RS-485 и протокола Modbus
^^^^^^^^^^^^^^^^^^

Раздел содержит параметры последовательного порта и параметры протокола Modbus Slave.
Параметры последовательного порта:

* включение/отключение интерфейса
* скорость передачи данных
* кол-во бит данных
* проверку чётности

Параметры Modbus Slave:

* Адрес Slave устройства в сети
* дополнительный тайм-аут ответа(для поддержки устаревших устройств, которые не могут немедленно приступить к 
  обработке ответа после выдачи запроса)
* тайм-аут опроса, по прошествии которого принимается решение о том, что отсутствует опрос со стороны *Мастера*

.. figure:: /img/web/rs485-1.png
       :align: center
       :alt: Настройка последовательного интерфейса RS-485 и протокола Modbus

       Настройка Modbus RS-485


Настройка интерфейса CAN и протокола CANopen
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Раздел содержит параметры *CAN* и параметры протокола *CANopen*:

* включение/отключение интерфейса
* скорость передачи данных
* *NodeID* в сети *CANopen*

       :align: center
       :alt: Настройка интерфейса CAN и протокола CANopen

       Настройка CANopen

Параметры линии электрообогрева
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Параметры линии электрообогрева разделены на группы:

* Текущие параметры
* Просмотр флагов ошибок и защитного отключения
* Настройки параметров линии
* Настройки аварийных пределов
* Настройка каналов ввода-вывода

Текущие параметры
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
Раздел используется для онлайн просмотра текущих параметров линии электрообогрева.
Содержит значения температур, токов, состояние выхода, текущий режим работы и другие параметры.
Позволяет управлять выходом линии в режиме дистанционного управления(в целях отладки).

.. figure:: /img/web/heating-live.png
       :align: center
       :alt: Просмотр текущих параметров линии электрообогрева

       Текущие параметры линии электрообогрева
