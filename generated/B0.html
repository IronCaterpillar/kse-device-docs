Длительность фильтра задаётся в миллисекундах. * Для плат 24В если более 5/6 выборок имеют логический уровень 1, то итоговое значение принимается равным 1, если менее 1/6 значений имеют логический уровень 1, то итоговое значение принимается равным 0, в промежуточном случае значение не меняется. * Для плат 220В если более 1/2 выборок имеют логический уровень 1, то итоговое значение принимается равным 1, если менее 1/10 значений имеют логический уровень 1, то итоговое значение принимается равным 0, в промежуточном случае значение не меняется.

Используется для перевода дискретного выхода в безопасное состояние в случае ошибки(см. subtable2012 ). Если значение параметра True, то параметр принимает значение в соответствии с 0x4n24 (см. B0 - DO Error Value ), где n принимает значение от 0 до 7, в зависимости от номера платы. Если же значение False, выход не меняет своего состояния.

10.20. B0 - AI Deadband¶

Субиндексы 0x3F35 B0 - AI Deadband(0—8)¶

Subindex

Название

Описание

Значение

Тип

Доступ

Сохранение

0

Number of Entries

Всего Субиндексов: 8

uint8

r

Нет

1

B0 - AI Deadband 1

0.1

float32

rw

Да

Зона нечувствительности задаётся в тысячных долях процента. После обработки значение записывается в 0x4n3A, 0x4n3B соответственно(subtable3F3A, B0 - AI Float Mag). Если значение превышает пределы, заданные в B0 - AI Upper Limit и B0 - AI Lower Limit, то значение в subtable3F3A и B0 - AI Float Mag меняется независимо от зоны нечувствительности.
