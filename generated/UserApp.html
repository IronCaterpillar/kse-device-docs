Внимание

Параметры Circuitbreaker OF, Circuitbreaker SD и Contactor отображают состояние объекта только если они соответствующим образом настроены в Line 1 HW Specific Settings

  • 1 - Fault

Данный параметр показывает, что присутствует по крайней мере один сигнал «Alarm» и/или «Trip». Расшифровку сигналов «Alarm» см. в Line 1 Temp Alarms, Line 1 Current Alarms, Line 1 Misc Alarms. Расшифровку сигналов «Trip» см. в Line 1 Temp Trip, Line 1 Current Trip.

  • 2 - Actual Control Mode

Режимы управления линией электрообогрева¶

Режим №

Описание

0

Нагрев всегда выключен

1

Нагрев всегда включен

2
Дистанционный режим

(управление через 0x5820:2)

см. Line 1 Control

3

Периодический (ШИМ) включение через заданные промежутки времени период - 0x5821:5, рабочий цикл, % - 0x5821:6 Line 1 Settings

  • 3 - Safe Mode On

Безопасный режим, в который линия электрообогрева переходит при наступлении следующих событий: * Ошибка расчёта температуры * Отсутствие опроса при работе в дистанционном режиме
