B2 - DI Filter Time 10

Время фильтрации вх. 10

uint8

rw

Да

Размерность время фильтра зависит оттипа платы дискретного ввода. Для платы 24В единица времени составляет 0,3мС, для платы 230В единица времени составляет 2мС. При вычислении значения сигнала происходит накопление значений на временном интервале фильтра. Если более 2/3 значений имеют логический уровень 1, то итоговое значение принимается равным 1, если менее 1/3 значений имеют логический уровень 1, то итоговое значение принимается равным 0, в промежуточном случае значение не меняется.
