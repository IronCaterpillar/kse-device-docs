1. Введение¶

Данное руководство содержит инструкцию по работе с устройством управления электрообогревом, его ПО, а так же объясняет принципы его функционирования, внутреннюю организацию и описание работы веб-интерфейса настройки и процесс обновления ПО.

1.1. Общие принципы работы¶

ПО устройства подразделяется на системное ПО и прикладное ПО. Настройка устройства выполняется с помощью встроенного в устройство WEB-интерфейса.

1.1.1. Системное ПО¶

Системное ПО отвечает за загрузку устройства, работу файловой системы, контроль состояния, индикацию, обработку каналов ввода-вывода, работу сетевых интерфейсов. Системное ПО может быть при необходимости обновлено для актуализации, либо для получения дополнительных возможностей, добавленных после выпуска устройства. Основой системного ПО является словарь объектов, где отображаются текущие данные, параметры и настройки устройства. Общее описание словаря: Устройство словаря объектов. Подробное содержание словаря: Словарь объектов. Словарь объектов является так же источником информации для отправки по интерфейсам связи. Для этого служат файлы настройки отображения. Каждому протоколу передачи данных соответствует свой файл/файлы настройки отображения, например, для протокола Modbus служит файл modbus_mappings.cfg. Описание процесса настройки отображения и настройку по умолчанию можно найти в соответствующих разделах.

1.1.2. Прикладное ПО¶

Прикладное ПО отвечает за задачу регулирования, контроля тока, защитное отключение в случае превышения допустимых пределов токов/температуры. Прикладное ПО устанавливается в процессе производства и может быть заменено независимо от системного ПО.
