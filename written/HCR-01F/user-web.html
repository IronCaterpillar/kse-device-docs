WEB-Интерфейс настройки устройства¶

Доступ к WEB-интерфейсу¶

При подключении устройства к компьютеру с помощью USB кабеля в системе должно появиться сетевое устройство. Драйвер в операционных системах Windows и Linux должен устанавливаться автоматически. Доступ к WEB-интерфейсу осуществляется по адресу http://169.254.241.1.

Страница быстрой настройки¶

Страница быстрой настройки предназначена для ускорения ввода устройства в эксплуатацию, а так же для упрощения работы по настройке устройства в процессе работы.

Общий вид страницы настройки¶

Страница настройки состоит из раздела управления автообновлением, раздела общих параметров, раздела настроек сетевых интерфейсов и протоколов, а так же блока управления индивидуальными параметрами устройства.

Общий вид страницы быстрой настройки¶

Раздел управления автообновлением¶

Предназначен для включения/отключения автообновления параметров устройства с помощью соответствующего переключателя, при включенном переключателе обновление выполняется каждые 2-3 секунды, при отключенном переключателе обновление происходит однократно при загрузке страницы. Раздел так же содержит информацию о текущем уровне доступа:

  • Off - контроль доступа отключен, полный доступ, можно менять любые параметры устройства

  • User - контроль доступа включен, вход не выполнен, доступ ограничен, можно менять только текущие оперативные параметры, настройки доступны только для чтения

  • Admin - контроль доступа включен, вход выполнен, полный доступ, можно изменять любые параметры

Раздел автообновления и состояния контроля доступа¶

Внимание

Автообновление может отключиться при потере связи с устройством. Если это произошло, необходимо вручную включить автообновление. В редких случаях при ошибках связи с устройством может потребоваться перезагрузка страницы(Ctrl-F5).

Раздел общих параметров¶

Данный раздел делится на область информации и область команд. Раздел информации содержит:

  • Код продукта

  • Уникальный номер устройства(UID)

  • Версия ПО

  • Версия прикладного приложения(при наличии)

Раздел команд содержит:

  • поле выбора и исполнения команды

  • поле ввода пароля

  • поле сброса к настройкам по умолчанию

Команды и управление доступом¶

Поддерживаются следующие команды:

  • Включение контроля доступа

  • Отключение контроля доступа

  • Сброс текущего уровня доступа

  • Перезагрузка устройства

Внимание

При включенном контроле доступа исполнение команд возможно только в режиме полного доступа после ввода пароля

Примечание

Для сброса к настройкам по умолчанию введите слово load в поле ввода и нажмите кнопку apply

Раздел настройки сетевых интерфейсов и протоколов¶

Настройка интерфейса RS-485 и протокола Modbus¶

Раздел содержит параметры последовательного порта и параметры протокола Modbus Slave. Параметры последовательного порта:

  • включение/отключение интерфейса

  • скорость передачи данных

  • проверка чётности

  • количество стоповых бит

Параметры Modbus Slave:

  • адрес Slave устройства в сети

  • дополнительный тайм-аут ответа(для поддержки устаревших устройств, которые не могут немедленно приступить к обработке ответа после выдачи запроса)

  • тайм-аут опроса, по прошествии которого принимается решение о том, что отсутствует опрос со стороны Мастера

Настройка Modbus RS-485¶

Настройка интерфейса CAN и протокола CANopen¶

Раздел содержит параметры CAN и параметры протокола CANopen:

  • включение/отключение интерфейса

  • скорость передачи данных

  • NodeID в сети CANopen

Настройка CANopen¶

Раздел параметров линии электрообогрева¶

Параметры в разделе линии электрообогрева разделены на следующие группы:

  • Текущие параметры

  • Просмотр флагов ошибок и защитного отключения

  • Настройки параметров линии

  • Настройки аварийных пределов

  • Настройка каналов ввода-вывода

Подробное описание параметров линии находится в соответствующем разделе описания словаря: Line 1 Description

Текущие параметры¶

Раздел используется для онлайн просмотра текущих параметров линии электрообогрева. Содержит значения температур, токов, состояние выхода, текущий режим работы и другие параметры. Позволяет управлять выходом линии в режиме дистанционного управления(в целях отладки). Так же в данном разделе осуществляется квитирование при аварийном отключении.

Текущие параметры линии электрообогрева¶

Просмотр флагов ошибок и защитного отключения¶

Раздел содержит флаги текущих активных ошибок, а так же флаги защитного отключения.

Флаги ошибок линии электрообогрева¶

Настройки аварийных пределов¶

Содержит настройки пределов сигнализации, а так же пределы защитного отключения. Параметр Alarm Hold Time служит для задания времени удержания флагов ошибок в случае их кратковременного возникновения и пропадания.

Настройка аварийных пределов¶

Настройки параметров линии¶

Содержит настройки линии: текущий режим управления, настройки режимов, режим расчёта температуры процесса, зоны нечувствительности для отправки аналоговых данных.

Настройка параметров линии¶

Настройка каналов ввода-вывода¶

Содержит настройки номеров каналов, настройки фильтрации аналоговых данных,а так же типы аналоговых датчиков.
